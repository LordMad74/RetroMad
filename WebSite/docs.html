Installation

Prérequis Système

  • OS : Windows 10 ou 11 (64-bit)
  • RAM : 4 Go minimum (8 Go recommandé)
  • GPU : Compatible OpenGL 3.0+ (pour RetroArch)
  • Espace Disque : 500 Mo pour l'application + espace pour les jeux
  • Dépendances : Visual C++ Redistributable, DirectX 9+

Étapes d'Installation

1 Téléchargement

Téléchargez la dernière version depuis GitHub Releases.

RetroMad_V0.7.0.7z (~85 Mo)

2 Extraction

Décompressez l'archive dans un dossier de votre choix (ex: C:\RetroMad).

Important : Évitez les chemins avec des espaces ou caractères spéciaux.

3 Lancement

Double-cliquez sur Lancer_RetroMad.ps1 (script PowerShell).

Si Windows bloque l'exécution, faites clic droit → "Exécuter avec PowerShell".

Interface Principale

Au premier démarrage, vous verrez 4 onglets principaux :

Constructeurs

Navigation par marque (Nintendo, Sega, Sony...)

Liste de tous vos systèmes configurés

Panneau d'administration complet

Configuration Initiale Recommandée

  1. Allez dans Admin → RetroArch
  2. Cliquez sur "Installer RetroArch (Stable)"
  3. Attendez la fin du téléchargement (~150 Mo)
  4. Cliquez sur "Installer Tous les Cores" pour les émulateurs
  5. Retournez dans Admin → Systèmes
  6. Ajoutez vos premiers systèmes (voir section suivante)

Démarrage Rapide (5 minutes)

Ajouter Votre Premier Système

1 Ouvrir le Gestionnaire

Admin → Systèmes → Bouton "+" (Nouveau Système)

2 Utiliser un Preset

Cliquez sur "Presets" et choisissez un système populaire (ex: NES, SNES, PlayStation).

Les presets configurent automatiquement : nom, core RetroArch, extensions de fichiers, et images.

3 Configurer le Dossier ROMs

Dans "Chemin ROMs", indiquez où se trouvent vos jeux (ex: D:\Roms\NES).

4 Sauvegarder

Cliquez sur le bouton "Sauvegarder" en haut à droite.

Lancer Votre Premier Jeu

  1. Retournez à l'onglet Consoles
  2. Cliquez sur la carte de votre système (ex: NES)
  3. Sélectionnez un jeu dans la liste
  4. Cliquez sur "Lancer" (icône Play)

Gestionnaire de Systèmes

Créer un Système Personnalisé

Informations de Base

  • ID : Identifiant unique (ex: "NES", "PSX")
  • Nom : Nom affiché (ex: "Nintendo Entertainment System")
  • Constructeur : Marque (Nintendo, Sega, Sony...)
  • Année : Année de sortie

Configuration Technique

  • Core RetroArch : Émulateur à utiliser (ex: "fceumm_libretro.dll" pour NES)
  • Extensions : Formats de fichiers acceptés (ex: ".nes, .zip")
  • Chemin ROMs : Dossier contenant les jeux

Médias

  • Image d'en-tête : Bannière principale (1920x1080 recommandé)
  • Logo : Logo du système (PNG transparent)

Recherche Automatique d'Images

Via Pexels (API)

  1. Allez dans Admin → Configuration
  2. Entrez votre clé API Pexels (gratuite sur pexels.com/api)
  3. Dans l'éditeur de système, cliquez sur "Web" (icône globe)
  4. Sélectionnez une image parmi les résultats

Via IA (Génération)

Fonctionnalité expérimentale. Nécessite une connexion internet.

Import Wikipedia

Pour remplir automatiquement les informations d'un système :

  1. Cliquez sur "Wikipedia" (icône W)
  2. Entrez le nom exact du système (ex: "Nintendo Entertainment System")
  3. Les champs seront pré-remplis avec les données extraites

Scraper (Skyscraper)

Qu'est-ce que le Scraping ?

Le scraping récupère automatiquement les métadonnées de vos jeux depuis des bases de données en ligne :

  • Jaquettes (boxart)
  • Screenshots
  • Vidéos de gameplay
  • Descriptions
  • Notes et dates de sortie

Configuration ScreenScraper

Créez un compte gratuit sur screenscraper.fr pour de meilleures performances.

  1. Ouvrez Skyscraper/config.ini
  2. Ajoutez vos identifiants ScreenScraper :
    [screenscraper]
    userCreds="votre_username:votre_password"

Lancer un Scraping

  1. Admin → Scraper
  2. Sélectionnez un système dans la liste
  3. Cliquez sur "Scraper"
  4. Attendez la fin du processus (logs en temps réel)
  5. Cliquez sur "Importer" pour intégrer les données

Le scraping peut prendre plusieurs minutes selon le nombre de jeux.

Configuration Mode Kiosk

Qu'est-ce que le Mode Kiosk ?

Le mode Kiosk transforme RetroMad en borne d'arcade professionnelle :

  • Interface plein écran verrouillée
  • Navigation uniquement au clavier/manette
  • Code PIN pour sortir
  • Économiseur d'écran automatique (Attract Mode)

Configuration

  1. Admin → Kiosk
  2. Choisissez un thème (Neon, Cyberpunk, Retrowave, Glass)
  3. Configurez le délai d'inactivité (secondes avant Attract Mode)
  4. Définissez un code PIN de sortie
  5. Activez/désactivez les effets sonores

Activer le Mode Kiosk

Depuis l'interface principale, cliquez sur l'icône (Monitor) en haut à droite.

Sortir du Mode Kiosk

Appuyez sur Échap 3 fois rapidement, puis entrez le code PIN.

WebPlay Mobile

Fonctionnement

WebPlay expose votre bibliothèque sur le réseau local via un serveur web intégré.

Connexion

  1. Assurez-vous que votre PC et mobile sont sur le même Wi-Fi
  2. Allez dans l'onglet WebPlay
  3. Scannez le QR Code avec votre téléphone
  4. Ou tapez l'adresse IP affichée dans votre navigateur mobile

Modes de Jeu

Mode Streaming (Par défaut)

  • Le jeu tourne sur votre PC
  • L'image est streamée vers votre mobile (MJPEG)
  • Latence minimale sur réseau local

Mode Local (WASM)

  • Le jeu tourne directement dans votre navigateur mobile
  • Utilise WebAssembly (Nostalgist/RetroArch)
  • Aucune latence réseau
  • Nécessite un smartphone puissant

Manettes

  • Tactile : Interface D-Pad + boutons avec retour haptique
  • Bluetooth : Connectez une manette PS4/Xbox à votre mobile

Lancer un Jeu

Depuis l'Interface Principale

  1. Sélectionnez un système (onglet Consoles)
  2. Cliquez sur un jeu dans la liste
  3. Une fiche détaillée s'affiche avec :
    • Jaquette
    • Description
    • Note
    • Date de sortie
  4. Cliquez sur le bouton "Lancer" (icône Play)

Options de Lancement

  • Lancer (Local) : Utilise RetroArch installé sur votre PC
  • Lancer (WebPlay) : Ouvre le jeu dans le navigateur mobile

Contrôles en Jeu

Les contrôles dépendent du core RetroArch utilisé. Par défaut :

  • F1 : Menu RetroArch
  • F2 : Sauvegarde rapide
  • F4 : Chargement rapide
  • Échap : Quitter le jeu

Gestionnaire de Sauvegardes

Créer un Backup

  1. Admin → Sauvegardes
  2. Cliquez sur "Créer une Sauvegarde"
  3. Un fichier .zip est créé avec :
    • Configuration RetroMad
    • Sauvegardes RetroArch (.srm)
    • Save states (.state)

Restaurer un Backup

  1. Sélectionnez un backup dans la liste
  2. Cliquez sur "Restaurer"
  3. Confirmez l'opération
  4. Redémarrez RetroMad

La restauration écrase les données actuelles. Créez un backup avant de restaurer !

Testeur de Manettes

Tester une Manette

  1. Connectez votre manette (USB ou Bluetooth)
  2. Admin → Manettes
  3. La manette est détectée automatiquement
  4. Appuyez sur les boutons pour voir la visualisation en temps réel

Manettes Supportées

  • Xbox One / Series (USB/Bluetooth)
  • PlayStation 4 / 5 (USB/Bluetooth)
  • Nintendo Switch Pro Controller
  • Manettes génériques DirectInput/XInput

Configuration dans RetroArch

RetroArch détecte automatiquement la plupart des manettes. Pour une configuration manuelle :

  1. Lancez un jeu
  2. Appuyez sur F1 (Menu RetroArch)
  3. Settings → Input → Port 1 Controls
  4. Mappez chaque bouton

Dépannage

RetroMad ne démarre pas

  • Vérifiez que PowerShell est autorisé à exécuter des scripts
  • Lancez en tant qu'administrateur
  • Vérifiez les logs dans logs/

RetroArch ne s'installe pas

  • Vérifiez votre connexion internet
  • Désactivez temporairement l'antivirus
  • Téléchargez manuellement depuis retroarch.com

Les jeux ne se lancent pas

  • Vérifiez que le core RetroArch est installé
  • Vérifiez le chemin des ROMs
  • Vérifiez les extensions de fichiers acceptées
  • Consultez les logs : Admin → Maintenance → Logs

WebPlay ne fonctionne pas

  • Vérifiez que PC et mobile sont sur le même réseau
  • Désactivez le pare-feu Windows temporairement
  • Vérifiez que le port 3000 n'est pas bloqué

Obtenir de l'Aide

Si le problème persiste :

  1. Consultez les logs : Admin → Maintenance
  2. Ouvrez une issue sur GitHub
  3. Incluez : version de RetroMad, OS, logs d'erreur
