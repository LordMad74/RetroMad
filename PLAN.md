# Projet Portail Retrogaming (RetroMad)

## Objectif
Créer un portail de retrogaming pour Windows et bornes d'arcade.
- Facade : Interface jolie (Console -> Jeux).
- Modes de jeu : Emulateurs locaux ou Webplayer (Réseau).
- Administration : Gestion des ROMs, émulateurs, images, thèmes.

## Architecture Technique
- **Frontend** : React + Vite (Rapide, moderne).
- **Design** : CSS moderne (Glassmorphism, animations) pour l'effet "Wow".
- **Application** : Electron (Pour l'intégration Windows/Arcade et lancement d'émulateurs).
- **Données** : Gestion locale (JSON/SQLite) pour les métadonnées.

## Étapes de Développement
1.  **Initialisation** : Structure du projet (Electron + Vite).
2.  **Interface Façade** : Design des systèmes et liste des jeux.
3.  **Administration** : Panel de gestion basique.
4.  **Intégration** : Lancement d'émulateur test.
